# Chelsea Performance Analysis

##  Project Overview
This project analyzes Chelsea match performance using PostgreSQL to identify win rate trends, home advantage impact, and possession influence on match results.

##  Business Questions
- What is Chelsea’s overall win rate?
- Does playing at home improve performance?
- Does higher possession increase winning probability?

## Tools Used
- PostgreSQL
- SQL

##  Key Insights
- Chelsea has a competitive overall win rate.
- Home matches show higher win rates compared to away matches.
- Higher possession correlates with improved match outcomes.
